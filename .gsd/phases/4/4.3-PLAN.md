# Plan 4.3: Real-time In-app Notifications

## Objective
Deliver real-time notifications for matches and team events to users while they are online, and display unread counts.

## Context
- .gsd/SPEC.md
- apps/api/src/routes/swipes.ts
- apps/web/src/components/dashboard/TopNavbar.tsx
- apps/web/src/components/notifications/NotificationBell.tsx (to create)

## Tasks

<task type="auto">
  <name>Broadcast Notifications from Backend</name>
  <files>apps/api/src/routes/swipes.ts</files>
  <action>
    - Import the Supabase client from `apps/api/src/lib/realtime.ts`.
    - Find the places where notifications are created via Prisma (e.g., in `swipes.ts` during match creation).
    - After Prisma creates the `Notification` record, broadcast it using `supabase.channel(\`notifications:\${userId}\`).send({ type: 'broadcast', event: 'new_notification', payload: notification })`.
  </action>
  <verify>grep -q "notifications:" apps/api/src/routes/swipes.ts && echo "Found broadcast"</verify>
  <done>Backend broadcasts new notification events instantly.</done>
</task>

<task type="auto">
  <name>Implement Real-time Notification Bell</name>
  <files>apps/web/src/components/notifications/NotificationBell.tsx, apps/web/src/components/dashboard/TopNavbar.tsx</files>
  <action>
    - Create `NotificationBell.tsx`.
    - On mount, it should fetch unread notifications via an API call (e.g., `GET /api/notifications` which was built in Phase 3/2).
    - It should subscribe to the broadcast channel `notifications:${currentUser.id}`.
    - On receiving a payload, increment the unread count and optionally show a toast/badge.
    - Integrate `NotificationBell` into the `TopNavbar.tsx` so it's visible globally across the dashboard.
  </action>
  <verify>npm run build -w web</verify>
  <done>NotificationBell displays real-time unread counts and updates instantly.</done>
</task>

## Success Criteria
- [ ] Swiping and matching triggers a real-time notification to the other user if they are online.
- [ ] Notification badge updates instantly without refreshing the page.
